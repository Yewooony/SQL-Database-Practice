# SQL-Database-Practice (극장 & 사원 DB 실습)
학교 데이터베이스 과목에서 진행한 SQL 실습 프로젝트 및 과제 모음입니다. 극장 및 사원 데이터베이스를 활용하여 DDL, DML, 고급 SELECT 쿼리 연습 결과를 포함하고 있습니다.

## 📌 프로젝트 개요 (Overview)

본 레포지토리는 [학교명]의 '[과목명]' 수업에서 진행한 **관계형 데이터베이스 및 SQL 실습 과제** 결과물입니다. 두 가지 주제의 데이터베이스를 직접 설계하고(DDL), 데이터를 삽입/수정하며(DML), 다양한 조건의 조회 쿼리(SELECT)를 작성하는 연습을 중점적으로 진행했습니다.

## 🛠️ 주요 실습 내용 (Key Practice Areas)

* **데이터 정의어 (DDL):** 테이블 생성 (`CREATE TABLE`), 제약조건(PRIMARY KEY, FOREIGN KEY, CHECK) 설정
* **데이터 조작어 (DML):** 데이터 삽입 (`INSERT`), 수정 (`UPDATE`)
* **SQL 질의어 (SELECT):**
    * **집계 질의 (Aggregate Queries):** `COUNT`, `SUM`, `AVG`, `MAX`, `MIN`
    * **조인 (JOIN):** INNER JOIN, LEFT JOIN을 활용한 다중 테이블 조회
    * **그룹화 (Grouping):** `GROUP BY` 및 `HAVING`을 사용한 조건부 그룹 질의
    * **부속 질의 (Subqueries):** 다른 쿼리의 결과를 활용한 질의
    * **뷰 (VIEW):** 복잡한 질의를 재활용하기 위한 뷰 생성 및 활용
